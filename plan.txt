# План: Добавление курицы с анимацией в JS Craft

## Описание задачи
Добавить в игру (файл `index.html`) моба **курицу** с анимацией (махание крыльями и хвостом при движении). Поведение — пассивное, как коровы (бродит, не агрессирует). Дропает мясо. Спавнится отдельным интервалом.

---

## 1. Создание текстуры курицы

Новая функция `createChickenTexture()` для генерации Canvas-текстуры курицы:
- Белое/светло-кремовое тело (Minecraft-стиль)
- Красный гребешок и бородка (wattle)
- Чёрные контуры ног и клюва

Материалы (добавить в `Materials`):
- `chickenBody` — белый/кремовый
- `chickenBeak` — оранжевый/жёлтый
- `chickenWattle` — красный
- `chickenLeg` — тёмно-розовый/бежевый

---

## 2. Геометрия курицы (`spawnChicken()`)

Курица строится из блоков (как корова и крипер), но компактнее:
- **Тело**: `BoxGeometry(0.4, 0.3, 0.5)` (плоское, вытянутое)
- **Голова**: `BoxGeometry(0.2, 0.25, 0.2)` (с поворотом)
- **Клюв**: маленький куб оранжевого цвета, выступает вперёд
- **Гребешок**: маленький красный куб на голове
- **Глаза**: чёрные точки на голове
- **Ноги**: 2 тонкие ноги (узкие блоки) с трансляцией вниз для шаговой анимации
- **Крылья**: 2 плоских крыла по бокам тела (позволяют анимировать взмах)
- **Хвост**: небольшой куб на задней части тела

Все части собираются в `THREE.Group`.

---

## 3. Анимация

В `updateMobs()` добавить блок `else if (mob.type === 'chicken')` с логикой хождения (похоже на корову).

**Анимация при ходьбе (targetAngle !== null):**
- `legSwing` увеличивается от `dt * 15`.
- Ноги вращаются по X с `Math.sin(legSwing)`: левая — forward, правая — backward.
- **Крылья**: mахают с большей частотой, чем ноги, создавая эффект «взмаха». `wingRotation.x = Math.sin(legSwing * 2) * 0.6` (широкий мах).
- **Тело**: поднимается-опускается на `Math.abs(Math.sin(legSwing)) * 0.03`, имитируя рывок при беге.
- **Хвост**: качается по X с `Math.cos(legSwing) * 0.15` (медленнее, чем крылья).
- **Голова**: лёгкое покачивание по X/Y.

**Анимация в покое (targetAngle === null):**
- Ноги возвращаются в нейтральное положение `lerp`.
- Крылья: `Math.sin(time * 3) * 0.1` — лёгкое подрагивание.
- Голова: вращает медленно из стороны в сторону, как курица осматривается.
- Шея/голова тянется вперёд с `Math.sin(time * 2) * 0.05` для эффекта «клёва земли».

---

## 4. ИИ и физика

- Курица бродит случайно, меняя направление (`targetAngle`) и таймер(
`moveTimer`), аналогично корове.
- Скорость ходьбы: без спринта, медленнее коровы (`~1.0` против `1.2`).
- Поворачивается плавно через `lerp` по `rotation.y`.
- Проверка столкновений (`checkCollision('mob')`) и прыжок при застревании (`v.y = 4` быльше чем у коровы `6`, потому что меньше).
- Гравитация и коллизии с блоками — общие (`mob.v.y -= 20 * dt`).
- При ударе мечом (в `handleAttack`): курица получает урон, дропает мясо при смерти, наносит частицы белого цвета.
- Если курица падает, также спавнит мясо.

---

## 5. Спавн и интеграция

- **Спавнер**: в `startGame` добавить `setInterval(spawnChicken, 5000);` (чуть чаще, чем корова `6000`, так как кур меньше).
- **Лимит**: максимум 8 куриц в мире.
- **Позиция спавна**: случайная точка вокруг камеры, как и у остальных мобов.

---

## 6. Дроп

- При гибели (в `updateMobs`, если hp <= 0 после handleAttack): 
  `spawnMeat(root.position)` — то же мясо что от коровы.
- При убийстве звук `hitMob`.
- При погибании частицы белые.

---

## 7. Модификации существующего кода

### `Materials`
Добавить:
```javascript
 Materials.chickenBody: new THREE.MeshLambertMaterial({ map: createChickenTexture() }), // или просто color: 0xfffdd0
 Materials.chickenBeak: new THREE.MeshLambertMaterial({ color: 0xffaa00 }),
 Materials.chickenWattle: new THREE.MeshLambertMaterial({ color: 0xcc0000 }),
 Materials.chickenLeg: new THREE.MeshLambertMaterial({ color: 0xcd853f }),
```

### `Entities.mobs` 
При спавне пушить объект `chicken` со струкурой, включающей ссылки на ноги, крылья, хвост, голову для анимации:
```javascript
Entities.mobs.push({ 
    g, type: 'chicken', hp: 2, v: new THREE.Vector3(), onGround: false, 
    targetAngle: Math.random() * Math.PI * 2, moveTimer: 0,
    legs: [leftLeg, rightLeg], 
    wings: [leftWing, rightWing],
    tail: tailMesh,
    head: headMesh,
    legSwing: 0,
});
```

### `updateMobs`
Добавить `else if (mob.type === 'chicken')` с логикой аналогичной корове, но с анимацией и частями.

### `startGame`
Добавить `Game.intervals.push(setInterval(spawnChicken, 5000));`.

### `handleAttack`
При splash/attacked внутри уже корректно, просто нужно чтобы hp <= 0 вызывало `spawnMeat(root.position)` для куриц и белые частицы, если курица убита.

---

## Примечания
- Текстура курицы создаётся через Canvas (без внешних изображений), чтобы оставаться self-contained.
- Все части тела — примитивы (Mesh) из BoxGeometry / PlaneGeometry, не используются внешние модели.
- Анимация крыльев и хвоста реализуется через обновление `rotation.x` в `gameLoop`.
